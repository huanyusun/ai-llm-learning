# 🧠 AI 大模型工程师 · 面试驱动项目式学习

> 理论 0 基础 → 通过**做项目**拿下 AI 工程师面试
> 工作流：**真实面试题 → 不会 → 做一个项目搞懂它 → 沉淀进仓库**

---

## 🎯 这是什么

一个以**面试题**为驱动、以**实战项目**为手段的 AI 大模型工程师学习仓库。

不是先啃理论再刷题，而是反过来：
1. 从真实的 AI 面试题集锦里**出一道题**
2. 不会？没关系——**直接动手做一个项目**，在做项目的过程中把背后的理论搞懂
3. 项目的代码、笔记、原题都提交进本仓库，随时同步 GitHub

> 为什么要这样学？因为面试考的是"你能不能讲清楚 + 做出来"，光背八股是讲不清楚的；而亲手做过一遍，理论自然就通了。
