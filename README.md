# 🧠 AI 大模型工程师 · 面试驱动项目式学习

> 理论 0 基础 → 通过**做项目**拿下 AI 工程师面试
> 工作流：**真实面试题 → 不会 → 做一个项目搞懂它 → 沉淀进仓库**

---

## 🎯 这是什么

一个以**面试题**为驱动、以**实战项目**为手段的 AI 大模型工程师学习仓库。

不是先啃理论再刷题，而是反过来：
1. 从真实的 AI 面试题集锦里**出一道题**
2. 不会？没关系——**直接动手做一个项目**，在做项目的过程中把背后的理论搞懂
3. 项目的代码、笔记、原题都提交进本仓库，随时同步 GitHub

> 为什么要这样学？因为面试考的是"你能不能讲清楚 + 做出来"，光背八股是讲不清楚的；而亲手做过一遍，理论自然就通了。

## 🎓 学习方法（基于系统知识库）

碎片化的文章学不透。本项目以**系统教材的知识体系**为底座：

1. **先建认知** → 读 [`knowledge/RAG/README.md`](knowledge/RAG/README.md) 导航 + 第1章，搞清 RAG 全貌
2. **系统学理论** → 按知识库章节顺序，每章末尾有【面试考点】【易错陷阱】
3. **动手做项目** → 每个知识点配真实项目（真实文档/检索/评估），做不懂回知识库查
4. **数学按需补** → 项目里用到什么数学（矩阵乘法/softmax/梯度），就在 `00-数学基础/` 补，不单独啃书

## 🔧 技术栈

- 向量库：Chroma / Milvus

> **运行项目**：`uv sync`（装依赖）→ `uv run python 脚本.py`（运行）
