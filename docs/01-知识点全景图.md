# 01 · AI 大模型工程师 · 知识点全景图

> 这是面试知识地图。每个模块都会用一个实战项目来学透。
> 🔥 数量代表面试热度。

---

## 一、数学与机器学习基础 🔥
- 线性代数（矩阵/向量/特征值）、概率统计（分布/贝叶斯/似然）
- 经典 ML：线性/逻辑回归、SVM、决策树、随机森林、XGBoost、聚类
- 梯度下降、过拟合与正则化（L1/L2/Dropout）
- 评估指标：精确率/召回率/F1/AUC-ROC/MSE

## 二、深度学习核心 🔥🔥
- 神经网络、反向传播
- 激活函数（ReLU/GELU/Sigmoid）、损失函数、优化器（SGD/Adam）
- CNN（卷积/池化）、RNN/LSTM

## 三、Transformer 与 NLP 🔥🔥🔥（核心）
- **Self-Attention 自注意力**、Multi-Head Attention
- 位置编码（RoPE 等）、Encoder-Decoder
- 预训练范式（BERT vs GPT）、Tokenizer（BPE/WordPiece）、词向量

## 四、大语言模型 LLM 🔥🔥🔥（核心）
- 自回归生成、**训练三阶段**（预训练 → SFT → RLHF/DPO）
- 上下文窗口、Token、采样参数（Temperature/Top-p/Top-k）
- 幻觉、**微调**（LoRA/QLoRA/PEFT/SFT）
- 推理优化（量化/KV Cache/Flash Attention/vLLM）、MoE

## 五、Prompt 工程 🔥🔥
- 基础提示、Few-shot、**CoT 思维链**、**ReAct**、结构化输出

## 六、RAG 检索增强 🔥🔥🔥（核心）
- 全流程：加载→分块→嵌入→向量库→检索→重排→生成
- 嵌入模型、向量数据库（Milvus/Chroma/Pinecone）、相似度
- 优化：Rerank/查询转换/混合检索/多粒度
- **RAG vs 微调怎么选**

## 七、Agent 智能体 🔥🔥🔥（核心）
- Agent 架构、**Function Calling 工具调用**、ReAct、Plan-and-Execute
- Multi-Agent、**MCP 协议**

## 八、开发框架与工具 🔥
- LangChain / LlamaIndex、HuggingFace Transformers、PyTorch 基础

## 九、应用与工程化 🔥
- 多模态、评估（RAGAS/忠实度/相关性）、部署、并发、流式输出、成本控制

## 十、场景设计与系统题 🔥🔥
- "如何设计一个智能客服""如何优化 RAG 系统""数据飞轮"

---

## 📌 学习顺序建议

```
Transformer(01) → LLM基础(02) → Prompt(03) → RAG(04) → Agent(05) → 微调(06)
     ↓理论地基        ↓必考常识      ↓最常用     ↓项目主力    ↓最热门    ↓进阶
```

**每个模块 = 一道面试题 + 一个实战项目 + 一份笔记**。
