---

## 为什么有这个知识库

碎片化文章学不透。这里把一本系统教材的**完整知识体系**结构化沉淀，作为 RAG 学习的**系统底座**：
- **学理论** → 来这里（概念、原理、流程、API）
- **做实战** → 去 `projects/` 或各模块文件夹（真实可运行项目）
- **备面试** → 每章末尾【面试考点】直接练

## 🎯 怎么用这个知识库

**学习路径（推荐顺序）**：
```
第1章(全局认知) → 第2章(环境) → 第3~7章(经典流水线，跟着代码跑) → 第8~13章(进阶，按需查)
```

**配合实战**：
- 知识库 = 理论底座（先读对应章节，建立理解）
- `04-RAG/projects/` = 真实项目（动手做，遇到不懂回这里查）

**面试冲刺**：
- 每章【面试考点】直接背 + 用自己的话复述
- 全景图见 [`docs/01-知识点全景图.md`](../../docs/01-知识点全景图.md)
- 真实面经见 [`docs/00-面试题资源汇总.md`](../../docs/00-面试题资源汇总.md)
