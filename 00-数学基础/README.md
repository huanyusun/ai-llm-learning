# 🔢 数学基础课程（AI 够用版）

> 从零开始，**只教 AI 用得到的数学**。每个概念都用 numpy + matplotlib **画出来**，让你"看见"数学，而不是背公式。
> 前置：会基本 Python（加减乘除、列表）。不需要任何数学基础。

---

## 学习方式

每节课 = 一个 `.py` 文件：
1. **概念**（用大白话 + 比喻）
2. **代码**（numpy 算出来）
3. **可视化**（matplotlib 画图）
4. **连回 AI**（这个数学在 AI 哪里用）

运行：`uv run python 00-数学基础/0X-xxx.py`，图存到 `img/`。

## 📐 第一部分：线性代数（AI 的语言）

| 课 | 主题 | AI 里的应用 |
|---|---|---|
| 01 | **向量**（加法/数乘/长度） | 词向量、特征、梯度都是向量 |
